<b>Um einen Server hinzuzufügen</b>:<br>
- Drücke auf den <img src="android:drawable/ic_menu_add"> Knopf in der Leiste oben.<br>
<br>
<b>Um einen Server zu bearbeiten oder zu löschen</b>:<br>
<br>
- Drücke lange auf denn Server denn du bearbeiten/löschen möchtest<br>
<br>
- Um mehrere Server zu makieren klicken sie diese einfach an. (Du kannst nur einen Bearbeiten)<br>
<br>
- Klicke auf denn Stift zum bearbeiten oder auf denn Mülleimer um denn makierten Server zu löschen<br>
<br>
Du kannst auf diese Informationen erneut zu greifen in dem du im Menu auf Hilfe klickst.<br>

<b>Um einen Server hinzuzufügen</b>:<br>
- Drücken sie auf <img src="android:drawable/ic_menu_add"> in der Leiste oben.<br>
<br>
<b>Um einen Server zu bearbeiten oder zu löschen</b>:<br>
<br>
- Drücken und halten sie lange denn gewünschten Server um diesen zu bearbeiten oder zu löschen<br>
<br>
- Um mehrere Server zu makieren klicken sie diese einfach an. (Sie können nur einen Server bearbeiten)<br>
<br>
- Klicken sie auf denn Stift um denn ausgewählten Server zu bearbeiten oder auf denn Mülleimer um diesen zu löschen<br>
<br>
Sie können auf diese Informationen erneut zu greifen in dem sie im Menü auf Hilfe klicken.<br>

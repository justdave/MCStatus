<short_description>
Ein Tool für Serveradmins um denn Status von Minecraft Servern zu verfolgen (Keine Chat App)
</short_description>
<full_description>
Mit dieser App kannst du schnell und leicht denn Status deines lieblings Minecraft Server verfolgen ohne Minecraft auf deinem Rechner öffnen zu müssen.

ACHTUNG: Das ist kein Spiel und keine Chat App. Dies ist ein Tool zum beobachten von Minecraft Servern, du benötigst weiterhin normales Minecraft, MineChat oder ähnliches um dem Server beizutreten.

Funktionen:

* Hinzufügen, löschen und/oder bearbeiten der Server in der Server Liste (Drück und halte einen Server um die Bearbeitungsoptionen zu öffnen)
* Zeigt dir folgende Informationen über die Server in der Liste:
* - Das Server Logo
* - Die Server MOTD (message of the day)
* - Wie viele Spieler auf dem Server sind und wie viel maximal drauf können
* - Die Version des Minecraft Servers
* - Wenn der Server es angibt, die Namen der vebundenen Spieler (Oder ein Teil davon bei großen Servern)

Die App funktioniert nur bei Servern die Minecraft 1.7 oder neuer unterstützen (da seit der 1.7 ein neues Server Ping Protocol verwendet wird)

Momentan musst du die Server Liste manuell aktualisieren (Drück dazu auf denn aktualisierungs Knopf in der oberen Leiste oder drehe deinen Bildschirm). Vielleicht aktualisieren ich die Server Liste automatisch während die App geöffnet ist, vielleicht lass ich die Liste auch im Hintergund aktualisieren um dann eine Benachrichtigung zu senden wenn jemand dem Server beitritt etc.

Die App ist Open-Source; Wenn du helfen möchtest, tu das bitte :-) Pull requests sind gerne gesehen. Das Projekt findest du auf GitHub https://github.com/justdave/MCStatus. Hier kannst du auch Bugs reporten oder neue Funktionen vorschlagen.

Für Entwickler: Die im Back-End benutzte Klasse um mit Server zu interargieren, ist so geschrieben das du sie einfach raus nehmen kannst um sie in deiner eigenen App zu verwenden. Wenn du dies tust, teile uns bitte alle Änderungen via GitHub mit, damit wir sie für alle zugänglich machen können.
